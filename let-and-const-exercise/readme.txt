Quiz
What is the difference between var and let? Var allows you to reassign and redeclare. Let only allows you to reassign. Var is hoisted. Let is only block scope.
What is the difference between var and const? Var allows you to reassign and redeclare. Const cannot reassign or redeclare.
What is the difference between let and const? Let allows you to reassign. Const does not.
What is hoisting? Hoisting is when variable names are added above their declared scope when the code is compiled. It means that their names are known from
  the beginning of the code but their values are not known until the code containing them is reached.
